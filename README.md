🔸 I built this dashboard as a personal exploration to simplify how satellite-derived Chlorophyll-a data can be accessed, visualized, and explored.

𝗧𝗵𝗲 𝗰𝗵𝗮𝗹𝗹𝗲𝗻𝗴𝗲:
🔸 Ocean colour analysis often involves multiple steps, data access, preprocessing, and switching between tools.

𝗪𝗵𝗮𝘁 𝘁𝗵𝗶𝘀 𝗱𝗮𝘀𝗵𝗯𝗼𝗮𝗿𝗱 𝗰𝗮𝗻 𝗱𝗼:
• Directly fetch ocean colour data via THREDDS / NCSS (no manual downloads)
 • Upload and parse both NetCDF-3 and NetCDF-4 files in-browser
 • Interactive AOI (Area of Interest) selection for focused analysis
 • Multi-variable support (Chlorophyll-a, kd490, reflectance, absorption, etc.)
 • Time-series animation and temporal navigation
 • Pixel-level inspection (value probing on map)

🔹 𝘼𝙙𝙫𝙖𝙣𝙘𝙚𝙙 𝙖𝙣𝙖𝙡𝙮𝙩𝙞𝙘𝙖𝙡 𝙛𝙚𝙖𝙩𝙪𝙧𝙚𝙨:
 • Transect analysis (profile extraction along a line)
 • Statistical summaries (min, max, mean, anomaly indicators)
 • Bloom/event detection support (via thresholds & trends)
 • Difference mapping (compare two datasets visually)
 • Correlation-style exploration (variable relationships)

🔹 𝙑𝙞𝙨𝙪𝙖𝙡𝙞𝙯𝙖𝙩𝙞𝙤𝙣 𝙘𝙤𝙣𝙩𝙧𝙤𝙡𝙨:
 • Log / linear scaling
 • Custom colour ramps
 • Opacity adjustment and overlay control
 • Land/cloud masking (fill value handling)

🔹 𝙐𝙨𝙚𝙧 𝙬𝙤𝙧𝙠𝙛𝙡𝙤𝙬 𝙚𝙣𝙝𝙖𝙣𝙘𝙚𝙢𝙚𝙣𝙩𝙨:
 • No dependency on external heavy software
 • Runs fully in-browser (lightweight and accessible)
 • Combines data access + visualization + basic analysis in one place

🔸 This is not a replacement for detailed scientific workflows; it supports faster exploration and preliminary understanding.

𝗗𝗮𝘁𝗮 𝘀𝗼𝘂𝗿𝗰𝗲𝘀:
 • ESA Ocean Colour CCI (OC-CCI)
 • THREDDS / NCSS services

𝗗𝗲𝘃𝗲𝗹𝗼𝗽𝗺𝗲𝗻𝘁 𝘀𝘁𝗮𝗰𝗸:
 • JavaScript (in-browser NetCDF handling)

 • Open-source ocean colour datasets
